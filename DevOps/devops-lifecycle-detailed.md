# DevOps Lifecycle Detailed Notes

## Plan

Tools: Jira, GitHub Issues, GitLab Issues, Confluence.

Outputs:

- User stories
- Acceptance criteria
- Architecture notes
- Sprint plan

## Code

Tools: Git, GitHub, GitLab, VS Code.

Outputs:

- Branches
- Commits
- Pull requests
- Code reviews

## Build

Tools: GitHub Actions, GitLab CI/CD, Jenkins.

Outputs:

- Build logs
- Test reports
- Artifacts
- Container images

## Test

Tools: unit test framework, Cypress, Playwright, Selenium, Postman.

Outputs:

- Test reports
- Coverage
- Quality gates

## Release

Tools: semantic versioning, release notes, artifact repositories.

Outputs:

- Versioned release
- Artifact checksum
- Deployment approval

## Deploy

Tools: Terraform, Ansible, Docker, Kubernetes, Helm, Argo CD.

Outputs:

- Infrastructure changes
- Application rollout
- Rollback plan

## Operate

Tools: Nginx, Kong, cloud load balancers, runbooks.

Outputs:

- Healthy traffic routing
- Incident response
- Capacity changes

## Monitor

Tools: Prometheus, Grafana, OpenTelemetry, ELK/OpenSearch, cloud monitoring.

Outputs:

- Metrics
- Logs
- Traces
- Alerts
- Postmortems
