# GCP Core Services Notes

## Structure

- Organization is the top-level boundary.
- Folders group projects.
- Projects contain resources and billing links.
- Service accounts are used by workloads.

## Core Services

- Compute Engine for VMs.
- Cloud Run for containerized apps.
- Cloud Storage for objects.
- Cloud SQL for relational databases.
- BigQuery for analytics.
- GKE for Kubernetes.
- Artifact Registry for images and packages.

## DevOps and Observability

- Cloud Build supports CI/CD.
- Cloud Deploy supports delivery workflows.
- Cloud Logging and Cloud Monitoring provide operations data.

## Practice

- Deploy Cloud Run service.
- Push image to Artifact Registry.
- Create GKE deployment.
- Create logs and alerts.
- Review billing and cleanup.
