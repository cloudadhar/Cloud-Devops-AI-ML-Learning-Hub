# Interview Question Bank

## Beginner

- Explain Git commit, branch, and pull request.
- Explain Linux file permissions.
- Explain DNS.
- Explain HTTP status codes.
- Explain Docker image and container.

## Intermediate

- Design a CI/CD pipeline for a web app.
- Explain Terraform state and drift.
- Explain Kubernetes service and ingress.
- Explain Nginx reverse proxy.
- Explain Kong API gateway.
- Explain Prometheus and Grafana.

## Advanced

- Design a secure multi-cloud deployment.
- Design a GitOps workflow.
- Design a DevSecOps pipeline.
- Design observability for microservices.
- Design an MLOps pipeline.
- Design a RAG application with evaluation and monitoring.

## Behavioral

- Tell me about a time you debugged a hard issue.
- Tell me about a time you learned a new tool quickly.
- Tell me about a time a deployment failed.
- Tell me about a time you improved documentation.
