## Announcements

Follow announcements here:

- [Announcements Discussion Category](https://github.com/cloudadhar/Cloud-Devops-AI-ML-Learning-Hub/discussions/categories/announcements)

Use announcements for:

- New roadmap updates
- New tool guide releases
- New lab additions
- Contribution campaigns
- Learning challenges
- Important repository maintenance updates

## What Belongs in Issues vs Discussions

| Need | Use |
| --- | --- |
| Broken link | GitHub Issue |
| Typo or documentation bug | GitHub Issue |
| New lab request | GitHub Issue |
| General learning question | GitHub Discussion |
| Announcement or community update | Announcements Discussion |
| Proposal for a large roadmap change | GitHub Discussion first, then pull request |

## Community Rule

Keep discussions learner-friendly, respectful, and practical. When sharing answers, include official documentation links whenever possible.
