# Foundation Knowledge Map

Before advanced Cloud, DevOps, MLOps, or AI work, learners need a foundation.

## Computer Basics

- CPU, memory, disk, network
- Process, thread, port
- File, directory, permission
- Environment variable
- Standard input, output, error

## Linux

- Filesystem hierarchy
- Users and groups
- Permissions
- Processes and services
- Logs
- Package managers
- Shell scripting

## Networking

- IP address and subnet
- DNS
- HTTP and HTTPS
- TCP and UDP
- Ports
- Routing
- NAT
- Firewall
- Load balancing

## Programming

Pick one main language:

- Python for automation, AI/ML, scripting, APIs
- JavaScript/TypeScript for web and full-stack apps

Learn:

- Variables
- Functions
- Loops
- Error handling
- Files
- APIs
- Testing

## Git

- Repository
- Commit
- Branch
- Merge
- Pull request
- Conflict resolution
- Tags and releases

## Documentation

Every learner should be able to write:

- Setup steps
- Architecture explanation
- Command explanation
- Troubleshooting notes
- Cleanup steps
