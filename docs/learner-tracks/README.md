## Interview Practice

Use [Interview Scenarios](interview-scenarios.md) after every topic. Do not memorize only definitions. Practice answering with:

```text
Problem -> Architecture -> Decision -> Tradeoff -> Security -> Monitoring -> Rollback
```

## How To Study

1. Pick one track.
2. Read the architecture mental model.
3. Finish the first 30-day plan.
4. Build one lab and capture screenshots/logs.
5. Answer the scenario questions in writing.
6. Add the project to your GitHub portfolio.
7. Move to the next track only after you can explain the first one clearly.
