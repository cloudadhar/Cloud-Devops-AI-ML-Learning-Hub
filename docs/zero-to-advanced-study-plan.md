# Zero to Advanced Study Plan

This plan is designed for 6 to 9 months of consistent learning. Move faster only if you can build and explain the projects.

## Phase 1: Foundation

Duration: 4 to 6 weeks

Learn:

- Computer basics
- Linux filesystem and commands
- Bash basics
- Git and GitHub
- Networking: IP, DNS, HTTP, TLS
- One programming language: Python or JavaScript

Build:

- Personal GitHub profile
- Linux command notes
- Static website
- Simple API

## Phase 2: Cloud Basics

Duration: 4 to 6 weeks

Learn:

- Cloud service models: IaaS, PaaS, SaaS
- AWS, Azure, and GCP basic services
- IAM and least privilege
- Compute, storage, networking, databases
- Billing alerts and cleanup

Build:

- VM-hosted website
- Object storage static site
- Basic VPC/VNet design
- Cloud cleanup checklist

## Phase 3: DevOps Core

Duration: 6 to 8 weeks

Learn:

- CI/CD
- Docker
- Nginx
- Terraform
- Ansible basics
- Artifact repositories
- Monitoring basics

Build:

- GitHub Actions pipeline
- Dockerized app
- Nginx reverse proxy
- Terraform EC2/VM deployment
- Basic Prometheus/Grafana dashboard

## Phase 4: Kubernetes and Platform

Duration: 6 to 8 weeks

Learn:

- Kubernetes pods, deployments, services, ingress
- Helm
- Argo CD
- Kong Gateway
- Secrets and config
- Resource requests and limits

Build:

- Kubernetes app deployment
- Helm chart
- Nginx ingress
- API gateway route with Kong
- GitOps deployment with Argo CD

## Phase 5: DevSecOps

Duration: 4 to 6 weeks

Learn:

- OWASP Top 10
- SAST, DAST, dependency scanning
- Container scanning
- IaC scanning
- Secrets scanning
- Vault and cloud secret stores

Build:

- Secure CI/CD pipeline
- Trivy scan
- Checkov scan
- SonarQube quality gate
- Threat model for your app

## Phase 6: AI/ML/MLOps

Duration: 8 to 10 weeks

Learn:

- Python data stack
- Model training and evaluation
- MLflow
- FastAPI model serving
- Docker and Kubernetes for model APIs
- RAG and vector databases
- Prompt evaluation and observability

Build:

- ML model API
- MLflow experiment tracking
- RAG chatbot
- AI API behind Nginx or Kong
- Monitoring dashboard for latency and errors

## Phase 7: Interview and Portfolio

Duration: ongoing

Prepare:

- Resume bullets with measurable proof
- GitHub portfolio
- Architecture diagrams
- Incident reports
- Interview answers
- Mock interviews
