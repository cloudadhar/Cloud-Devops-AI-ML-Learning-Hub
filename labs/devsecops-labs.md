# DevSecOps Labs

## Lab 1: Secret Scanning

Goal: prevent secrets from entering Git.

Tools:

- Gitleaks
- GitHub secret scanning where available

## Lab 2: Container Scan

Goal: scan Docker image for vulnerabilities.

Tool:

- Trivy

Questions:

- What is a CVE?
- What do you do if a base image is vulnerable?

## Lab 3: IaC Scan

Goal: scan Terraform for risky cloud configuration.

Tools:

- Checkov
- tfsec

## Lab 4: API Security Gateway

Goal: use Kong to add API key authentication and rate limiting.

Questions:

- What is authentication?
- What is authorization?
- Why rate limit APIs?

## Lab 5: OWASP Top 10 Review

Goal: map a sample app against OWASP risks and write fixes.
