## How To Use A Project

1. Read the problem statement and architecture.
2. Run the project locally.
3. Change one feature yourself.
4. Add logs, tests, or security controls.
5. Record screenshots or terminal output.
6. Explain the architecture in your own words.
7. Add a short demo video or LinkedIn post.
