# Project Rubric

Use this rubric to judge whether a learner project is portfolio-ready.

## Basic Requirements

- Clear problem statement.
- Working code.
- README with setup steps.
- Screenshots or output.
- Git history.

## DevOps Requirements

- CI/CD pipeline.
- Dockerfile if applicable.
- Environment variables documented.
- Deployment steps.
- Cleanup steps.

## Security Requirements

- No secrets committed.
- Dependencies reviewed.
- Public ports explained.
- IAM permissions documented.

## Observability Requirements

- Logs available.
- Health endpoint or status check.
- Metrics or dashboard for advanced projects.

## Interview Readiness

Learner can explain:

- Architecture
- Tool choices
- Failure cases
- Security controls
- Monitoring
- Cost and cleanup
