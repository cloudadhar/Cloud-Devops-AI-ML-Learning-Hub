## Recommended Study Order

1. Linux, networking, Git and GitHub.
2. One role track from [Learner Tracks](../docs/learner-tracks/README.md).
3. One cloud provider: AWS, Azure, or GCP.
4. CI/CD: GitHub Actions, GitLab CI, or Jenkins.
5. Docker, Nginx, and API basics.
6. Terraform and Ansible.
7. Kubernetes, Helm, and Argo CD.
8. Prometheus, Grafana, and OpenTelemetry.
9. SonarQube, Trivy, Checkov, Nexus, Artifactory.
10. AI, ML, MLOps, RAG, or Agentic AI only when the chosen role requires it.
