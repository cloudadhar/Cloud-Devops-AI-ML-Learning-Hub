## Recommended Study Order

1. Linux, Networking, Git and GitHub.
2. One cloud provider: AWS, Azure, or GCP.
3. GitHub Actions or GitLab CI/CD.
4. Docker and Nginx.
5. Terraform and Ansible.
6. Kubernetes, Helm, and Argo CD.
7. Prometheus, Grafana, OpenTelemetry.
8. SonarQube, Trivy, Checkov, Nexus or Artifactory.
9. Kong Gateway for API platforms.
10. AI/ML, MLOps, RAG, and Agentic AI.
